FEATURED PROJECT


DockIt

DockIt is a solo project based on the request of a client whose nationally-based restoration company needed a millennial-ized CRM to manage assignments from start to finish. In less than 2 weeks, I presented a mobile responsive, custom CRM with full user authentication and authorization with three distinct roles (basic user, manager, and admin).

UX-First Core Functionality

The main dock has 4 essential tools designed to keep assignments organized and ensure responsibilities are managed in a timely manner.

  • To-Do List
  • Assignments
  • Resources
  • Bulletin Board

I wanted to ensure the best possible user experience, so I worked closely with the client to identify how to best display tons of information without cluttering the screen. I made this magic happen with a lot of collapsible content and a front end framework I like to call "Fangular."

Fangular

The client stressed the importance of having everything on one page, but having built the app with Rails, the convention was to use a multi-page format. I manipulated the hell out of it with custom JavaScript functions to impersonate (or fake) a client-side MVC framework like AngularJS...hence, "Fangular."
